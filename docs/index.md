# Currency Converter

Convert USD to SGD — and between any major currencies — using live reference rates. A free, zero-dependency Web Component you can drop into any site.

```html
<script src="currency-converter.js"></script>
<currency-converter></currency-converter>
```

## What it does

This converter fetches live mid-market reference rates (European Central Bank rates) and converts instantly between major currencies including SGD, USD, MYR, INR, EUR and GBP. The rate and its date are shown with every result.

## When you'd use it

Quoting an overseas client, estimating a foreign invoice, or pricing a product for another market.

See the [Usage & API guide](usage.md) for framework examples, or the [guide](guide.md) for tips.
