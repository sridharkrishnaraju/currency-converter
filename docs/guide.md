# Guide: Reference rates vs the rate you pay

These are mid-market reference rates — the 'true' rate you see on Google. When you actually exchange money, banks and payment providers add a spread, so the rate you pay is a little worse. For a Singapore SME quoting in USD, build that margin into your pricing so an FX move doesn't eat your profit.
