# Currency Converter

Convert USD to SGD — and between any major currencies — using live reference rates. Zero dependencies — works in plain HTML, React, Vue, Svelte or Astro.

**▶ [Live demo](https://sgbp.tech/tools/currency-converter)**

```html
<script src="currency-converter.js"></script>
<currency-converter></currency-converter>
```

## What it does

This converter fetches live mid-market reference rates (European Central Bank rates) and converts instantly between major currencies including SGD, USD, MYR, INR, EUR and GBP. The rate and its date are shown with every result.

## Install

```bash
npm install @sgbp/currency-converter
```

or copy `currency-converter.js` into your project.
